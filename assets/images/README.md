# ATMOS-TRS logo

This folder contains **logo.png**, used for:

- **Login screen** – orange header
- **Tourism dashboard** – drawer/sidebar beside "ATMOS TRS" and "Tourism Office"
- **Governor dashboard** – sidebar branding
- **App launcher icon** – icon on home screen after install (Android & iOS)

To change the logo: replace **logo.png** with your image, then run:
```bash
dart run flutter_launcher_icons
```
Then rebuild/reinstall the app.
